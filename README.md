## 📱 Features  
- **Add/Delete tasks** with simple UI  
- **Local storage** using SharedPreferences  
- **No complex state management** (setState only)  
- Minimalist design  

## 🛠 Tech Stack  
```yaml
dependencies:
  flutter:
    sdk: flutter
  shared_preferences: ^2.2.2
```
##🚀 Installation
```bash
git clone https://github.com/pamiracar/Task-Riot.git
cd Task-Riot
flutter pub get
flutter run
```
